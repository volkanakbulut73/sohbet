
WORKIGOM CHAT - GITHUB & HOSTINGER DEPLOYMENT REHBERİ (v1.0.8)

Bu proje şu an Otomatik Dağıtım (Auto-Deploy) için hazır hale getirilmiştir.

ADIM 1: GITHUB WEBHOOK AYARI (KRİTİK)
-------------------------------------
GitHub reponuzda (https://github.com/volkanakbulut73/sohbet) şu adımları takip edin:
1. Reponuzun "Settings" -> "Webhooks" sekmesine gidin.
2. "Add webhook" butonuna tıklayın.
3. "Payload URL" kısmına şu adresi yapıştırın:
   https://webhooks.hostinger.com/deploy/4c6b71374c5b525227208722f38ac5f5
4. "Content type" olarak "application/json" seçin.
5. "Which events..." kısmında "Just the push event" seçili kalsın.
6. "Add webhook" diyerek kaydedin.

ADIM 2: HOSTINGER HPANEL YAPILANDIRMASI
---------------------------------------
1. Hostinger hPanel -> Websites -> www.workigomchat.online -> Manage.
2. "Advanced" -> "GIT" sekmesine gidin (Veya "Front-end" deployment aracını kullanın).
3. "Deployment Directory" olarak "/" (veya build alıyorsanız "/dist") belirleyin.
4. Artık bilgisayarınızdan reponuza kod gönderdiğinizde, Hostinger bu değişikliği algılayıp siteyi güncelleyecektir.

ADIM 3: BUILD İŞLEMİ (Opsiyonel)
-------------------------------
Eğer Hostinger'da "Front-end" aracını kullanıyorsanız, Hostinger sizin yerinize "npm install && npm run build" komutlarını çalıştıracaktır. Eğer manuel FTP kullanacaksanız, sadece "dist" klasörünün içindekileri yükleyin.

DİKKAT:
Herhangi bir hata durumunda Hostinger hPanel -> "Log Files" kısmından webhook'un başarıyla tetiklenip tetiklenmediğini görebilirsiniz.
